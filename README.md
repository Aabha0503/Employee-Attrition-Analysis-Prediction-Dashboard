# Employee-Attrition-Analysis-Prediction-Dashboard

## Overview
This project focuses on analyzing employee attrition patterns and predicting employee turnover using Machine Learning techniques. The system helps identify key factors influencing attrition and provides data-driven insights for HR decision-making.

The project combines Exploratory Data Analysis (EDA), visualization, and predictive modeling to analyze employee behavior and workforce trends.

---

## Features
- Employee attrition analysis and visualization
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Random Forest Classification model
- Attrition prediction system
- Confusion Matrix evaluation
- Interactive charts and dashboards
- HR insights and KPI analysis
